# Fullstack Test

Este proyecto fue desarrollado con **Angular 19** para el frontend y **Spring Boot 3.2.6** para el backend. A continuación una guía rápida para configurar y ejecutar el proyecto.

## Estructura del Proyecto
- **Frontend**: Aplicación Angular 19. [Ver README del frontend](./frontend/README.md).
- **Backend**: API REST con Spring Boot 3.2.6. [Ver README del backend](./backend/README.md).

## Requisitos Previos
- **Frontend**: Node.js 18+ y Angular CLI 19.1.5.
- **Backend**: Java 17, SQL Server y Maven.

## Configuración Rápida
1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/fullstack-test.git
   cd fullstack-test
2. Configura y ejecuta el backend:

    ```bash
    cd backend
    mvn spring-boot:run
3. Configura y ejecuta el frontend:

    ```bash
    cd ../frontend
    npm install
    ng serve
